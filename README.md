# Allied Immigration — Website

Official website for **Allied Immigration**, a CICC-certified Canadian immigration consultancy serving bilingual (English/Chinese) clients worldwide.

**Live site:** https://luoxiaobin.github.io/awcanada-new/

---

## About

This site helps clients navigate Canadian immigration pathways including Express Entry, Family Sponsorship, Study Visas, and Work Permits. It features a bilingual interface (English/Chinese), an eligibility quiz for lead capture, and direct contact via WeChat and WhatsApp.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** (build tool)
- **Lucide React** (icons)
